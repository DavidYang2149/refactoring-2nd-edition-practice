# refactoring-2nd-edition-practice

- 리팩터링 2판 - 코드 예제 연습하기
- index.js 코드를 리팩터링 해보세요!
- 9장 ~ 10장 내용을 연습해보기
- 각자 브랜치로 Pull Request 후 리뷰 받아보기
- 다른 Pull Request를 리뷰해보기
- Pull Request 성공하기 (조건: 리뷰 1개 필수)
## 📚 Install dependencies

```sh
npm install
```

## Run

```sh
# node version v14 이상
npm start
```

## Run Coverage

```sh
# basic test
npm run coverage

# running option
npx test
```
